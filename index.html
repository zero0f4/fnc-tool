JSCU Logging Essentials
Joint Sigint Cyber Unit Nederland — Windows Event Logging Baseline
GitHub
Over deze baseline
De JSCU Logging Essentials biedt hands-on richtlijnen voor het configureren van Windows Event Logging en het centraal verzamelen via Windows Event Forwarding (WEF/WEC). De baseline bevat 194 event IDs verdeeld over 16 categorieën die detectie- en forensische waarde hebben met minimale ruis. Complementair aan AV-, IDS- en EDR-oplossingen. Minimaal Windows 10 1709 / Server 1803. Aanbevolen retentie: minimaal 6 maanden.
